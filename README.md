# Website com NodeJs
Praticando criação de site com Node + Typescript + Mustache

### Pré-requisitos globais:
`npm i -g nodemon typescript ts-node`

### Instalação
`npm install`

### Para rodar o projeto
`npm run start-dev`

### Objetivo

O objetivo desse projeto é de testar minhas habilidades de programação utilzando as novas linguagens de programação que aprendi recentemente, é muito bom sempre evoluir e aprender cada vez mais.
